title: 'System Health'
description: 'Monitor system health and status'

## Health Check

### Endpoint

```
GET /health
```

### Authentication

Not required

### Response

```json
{
  "status": "healthy",
  "timestamp": "2026-01-16T10:30:00Z",
  "version": "1.0.0",
  "components": {
    "api": "healthy",
    "database": "healthy",
    "redis": "healthy"
  },
  "uptime_seconds": 86400
}
```

### Status Values

| Status | Description |
|--------|-------------|
| `healthy` | All systems operational |
| `degraded` | Some components impaired but functional |
| `unhealthy` | Critical components failing |
